README
***********

To build and run the program:

	Use the makefile in the src directory. Run 'make' to compile all the files. Run 'make run' to initialize and run the program.


program requirements:

	To run the program, the newest version of java must be installed on the system, Java 7.


The classes to look into are the Module class, the processSimulator class, and the site class. to test it simply go to modules tab, enter a number for module time estimation, click add module, then click simulate end of day (this will run endofdaysim method in process simulator). You can add multiple modules also. You got to edit end ofdaysim method and add methods to actually run the process simulator
